# 🎨 AI Image Generator Pro

A modern AI Image Generator built with **Python** and **Streamlit**.

Generate beautiful AI images from text prompts and download them instantly.

---

## ✨ Features

- 🎨 AI Image Generation
- 🌙 Modern Dark UI
- 📱 Mobile Friendly
- ⚡ Fast Generation
- 🖼️ Multiple Images
- 📥 Download PNG Images
- 📦 Download All Images as ZIP
- ☁️ Streamlit Cloud Ready

---

## 🛠️ Built With

- Python
- Streamlit
- Requests
- Pillow

---

## 🚀 Installation

```bash
pip install -r requirements.txt
streamlit run app.py
```

---

## 📸 Preview

Generate images from your imagination using simple text prompts.
